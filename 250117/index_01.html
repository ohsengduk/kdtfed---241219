* 참조 타입의 변수
      객체 영역에 있는 타입을 참조해서 가져 온다

      원시 타입은 본연의 속성 값을 그대로 사용하기 때문에 원시타입이라고 한다( 스트링 넘버등을 말한다 )

      let a = 10;
      let b = a;

      console.log(a, b);

      let obj1 = {
        c: 10,
        d: "ddd",
      };

      let obj2 = obj1;

      console.log(obj1, obj2);

      b = 15;
      obj2.c = 20;
      console.log(a, b);

      console.log(obj1, obj2);

      전개연산자 구문(스프레드 오페레이드 구문)
